# Handoff Archive

> `TASK_LIST.md`에서 밀려난 오래된 handoff 원문을 보관하는 문서입니다.  
> 기본 진입 문서가 아니며, 충돌 복구 / version closeout / 회고가 필요할 때만 읽습니다.

## Usage Rules
- 기본 작업 시작 시 이 문서를 읽지 않습니다.
- 오래된 항목만 이 문서로 이동합니다.
- 항목 순서는 오래된 것이 위, 최신 archive 항목이 아래입니다.
- `CURRENT_STATE.md`와 `TASK_LIST.md`를 대체하지 않습니다.
